# GAS ドキュメントビューアー

Googleドライブの指定フォルダ内にあるGoogleドキュメントを、左側に垂直タブのファイル一覧・右側にドキュメント本体（iframe）として表示するカスタムビューアー。

## セットアップ手順

### 1. GASプロジェクトを作成
- [Google Apps Script](https://script.google.com/) を開き、新しいプロジェクトを作成

### 2. ファイルを配置
- `Code.gs` の内容をデフォルトの `コード.gs` に貼り付け
- `ファイル > + > HTML` から `index` という名前でHTMLファイルを作成し、`index.html` の内容を貼り付け

### 3. フォルダIDを設定
- `Code.gs` の `FOLDER_ID` を対象フォルダのIDに書き換える
- フォルダIDはGoogleドライブでフォルダを開いた時のURLの末尾部分
  - 例: `https://drive.google.com/drive/folders/1aBcDeFgHiJkLmNoPqRs` → `1aBcDeFgHiJkLmNoPqRs`

### 4. デプロイ
- `デプロイ > 新しいデプロイ` を選択
- 種類: `ウェブアプリ`
- 次のユーザーとして実行: `自分`
- アクセスできるユーザー: 組織内 or 全員（用途に応じて）

## 機能
- 📁 フォルダの再帰的ツリー表示（アコーディオン開閉）
- 🔍 ファイル名によるリアルタイム検索フィルター
- 📄 Googleドキュメントのプレビュー表示（iframe）
- 🎨 ダークテーマのサイドバー

## 注意事項
- ドキュメントのURLは自動的に `/preview` 形式に変換される
- 閲覧権限のないユーザーにはドキュメントが表示されない（ドライブの権限に依存）
- 大量のファイル（数百件以上）がある場合、初回読み込みに時間がかかる可能性あり
